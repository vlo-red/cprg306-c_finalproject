# Project Title

ClimbR

---

## Description

This is a next.js web app that will help users to log routes climbed at a specific gym, record climbing sessions, as well as review their climbing history to monitor their progress over time.

---

## Installation Instructions

Follow these steps to run the project locally:

```bash
git clone https://github.com/vlo-red/cprg306-c_finalproject
cd cprg306-c_finalproject
npm install
```

---

## Usage

Start the development server:
```bash
npm run dev
```

Then open your browser and go to:
`http://localhost:3000`

From there, you can use the application by navigating through pages and interacting with the features.
